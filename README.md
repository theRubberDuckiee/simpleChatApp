## How to run this project
1. Clone this project
1. Open up terminal and run `yarn`
1. Run `npm run api-server`
2. Open up another terminal window and run `yarn run start`
3. This should automatically open up a window with the web app running.
4. You can log in with a username and start chatting. If you want to log in as another user, simply refresh the page.
